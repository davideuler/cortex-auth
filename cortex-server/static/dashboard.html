Shamir m-of-n share generation
⚠️
This splits the running KEK into n shares such that any m of them can recover it. Distribute the shares to the operators NOW — the server does not retain them. To recover a lost-password installation, restart with CORTEX_RECOVERY_MODE=1 and CORTEX_RECOVERY_THRESHOLD=<m>; the server will prompt for shares on stdin.
